//to install the requirement
- install phython //Python 3.10.8 version
- pip install -r requirements.txt // to install requirements from the requirements.txt ( my current versions: torch(2.1.2), torchaudio(2.5.1+cu118), faster-whisper(1.0.0), transformers(4.39.3), pandas(2.2.3), setuptools(75.5.0), nltk(3.9.1) )
- install "CUDA Toolkit" version 12.6 from nvidia
- pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118 // this version worked for me latest one are bugged and these are nvidia cuda drivers

//Happy face
- in .env but happy face api from https://huggingface.co

//how to run this 
python -m venv whisperx-env // do this when there is no whisperx-env folder, upon completion whisperx-env will be made and then run the below command.
whisperx-env\Scripts\activate // do this when there is whisperx-env folder, upon completion terminal command will start with (active) and then run the next command
phython main.py
